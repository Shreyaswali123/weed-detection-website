How It Works

1

Upload Image

Upload any field photograph — drone, handheld camera, or mobile. Supports JPG, PNG, WEBP formats.

2

YOLOv8 Inference

Your image is sent to the FastAPI backend where our trained YOLOv8 model runs real-time object detection.

3

Results & Bounding Boxes

Detected objects are returned with bounding boxes, class labels, and confidence scores for each detection.

🔌 API Integration

Integrate weed detection directly into your application with a simple POST request.

# POST /predict — send an image, get detections curl -X POST "http://localhost:8000/predict" \ -F "file=@field_photo.jpg" \ | python3 -m json.tool # Example response: { "detections": [ { "class_name": "weed", "confidence": 0.8934, "bbox": [120, 45, 310, 200] } ], "weed_count": 3, "crop_count": 7, "inference_time_ms": 18.4 }
